What is Eztables?

Eztables allows you to setup a firewall on a Linux system with human-friendly rules. Eztables manages the iptables rules for you, so you don't have to. The rules you have to write, are in a human-friendly, easy to understand format.

Example

Just take a look at this example of a simple home or small-business networking setup:

    nat $eth1_net $eth0
    allow_forward $eth1_net any any any
    allow_in $eth1_net $eth1 any 22/tcp

You probably can figure out with ease what these rules do. The first one sets up NAT. The second one allows traffic to flow from within your network to the Internet and the last one allows SSH access to the firewall itself.

Any other benefits?

Eztables offers a feature found in most commercial firewalls: it allows you to work with objects or groups. Take a look:

    HTTP_SERVICES="
        80/tcp
       443/tcp
    "

    WEB_SERVER_1=192.168.0.10
    WEB_SERVER_2=192.168.0.11

    WEB_SERVERS="
        $WEB_SERVER_1
        $WEB_SERVER_2
    "

    allow_in any "$WEB_SERVERS" any "$HTTP_SERVICES"

By defining groups of hosts, networks and services, you can keep the number firewall rules very small. This way, you don't get overwhelmed with rules and there is less risk of a typo creating an insecure configuration. You stay in control because you understand your rule base with a quick glance.

Features

Eztables is designed to be simple, yet powerful. It doesn't matter if you are setting up a home router, or use it to setup a corporate firewall. Eztables supports:

  • Basic input / output filtering
  • Network address translation (NAT)
  • Port address translation
  • Support for vlans
  • Service and host groups / objects
  • Logging to syslog
  • Support for plugins
  • Automatically detects all network interfaces
